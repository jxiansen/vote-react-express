# React-vote

### 依赖模块

本项目使用 `vite` 创建，具体开发过程使用了以下依赖包

* `react`
* `react-router`: 路由导航
* `antd-mobile` : 蚂蚁金服开源的 `react` 移动端 `ui` 框架
* `axios`: 网络请求库，前后端交互使用
* `immer`: `mobx` 作者写的一个库，用来实现 `js` 的不可变数据结构
* `typescript`

## 功能

* 创建投票（单选投票，多选投票）
* 用户登录
* 投票详情查看
* 编辑投票
